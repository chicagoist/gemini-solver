

###  Установка .xpi в Firefox

Тут есть подвох. Обычная версия Firefox **запрещает** устанавливать неподписанные `.xpi` файлы (защита от вирусов).

У тебя есть два пути:

**Путь 1. Для разработки (Временный, рекомендуемый)**
Ты уже это делал. Не используй `.xpi`, используй загрузку папки.

1.  `about:debugging` -\> "Этот Firefox".
2.  "Загрузить временное дополнение".
3.  Выбираешь `manifest.json`.
    *Минус:* Слелетает после перезапуска браузера.

**Путь 2. Для постоянного использования (Через .xpi)**
Чтобы установить `.xpi` навсегда, его нужно "подписать" у Mozilla или использовать специальную версию браузера.

1.  Скачай **Firefox Developer Edition** (или Firefox Nightly). В обычной версии это не сработает.
2.  В Developer Edition зайди в `about:config`.
3.  Найди параметр `xpinstall.signatures.required` и поставь его в **false**.
5. Найди параметр `extensions.langpacks.signatures.required` и поставь его в **false**.
6.  Теперь просто перетащи свой файл `gemini-solver.xpi` в окно браузера. 
6.  Firefox спросит: "Добавить Gemini Exam Solver?" -\> Жми "Добавить".

Теперь расширение установлено как обычный плагин и не исчезнет после перезагрузки.
